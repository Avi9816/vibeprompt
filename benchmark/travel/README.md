# Travel Benchmarks

Use this folder for travel, location, street, landscape, transit, hotel, and destination benchmark cases.

## Workflow

- Record the original reel reference privately or use a rights-cleared sample.
- Save extracted JSON for factual and semantic analysis.
- Save generated master and platform prompts.
- Save generated video result references when publication rights allow.
- Add rating notes, issue tags, and comparison observations.

Do not commit copyrighted videos, private reels, or generated media without permission.
