# Changelog

## Unreleased

### Platform Prompt Templates

Added platform-specific prompt structures for Veo, Sora, Runway, Kling, and Pika so outputs can follow each model's preferred style and emphasis.

### Speech Confidence

Added speech confidence scoring, transcript quality checks, multilingual language handling, and guardrails that prevent weak transcripts from dominating semantic understanding.

### Audio Intelligence

Added audio type, dialogue summary, music mood, ambient audio, and prompt integration rules for speech, music, and mixed audio cases.

### Motion Energy Intelligence

Added motion energy fields for dance, fitness, sports, and high-energy social-media content, with grounded intensity and rhythm descriptions.

### Creator Archetype Intelligence

Added creator archetype detection for beauty, fashion, fitness, food, educational, tech, and lifestyle creator styles.

### Temporal Progression

Added temporal reel progression fields to make prompts describe evolving moments rather than static visual summaries.

### Attention Direction

Added attention direction fields for primary focus, secondary focus, focal transitions, and visual priority flow.

### Visual Translation

Added generative visual language translation so internal semantic intelligence becomes direct cinematic prompt language.

### Prompt Guardrails

Added prompt naturalness checks and rollback behavior to prevent optimization layers from producing robotic or repetitive text.

### Feedback Optimization

Added prompt feedback storage, analytics, and feedback-aware optimization hooks for long-term prompt quality improvement.
